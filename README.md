# 英語論文PDF 日本語翻訳ビューア / PDF再生成アプリ

GitHub Pages で配布可能な、完全クライアントサイドのPDF翻訳SPAです。

## 機能

- PDFアップロード（ファイル選択 / D&D）
- 10MB上限バリデーション、20ページ超の警告
- テキスト抽出可能PDF判定（スキャンPDFの簡易除外）
- 文書構造推定（タイトル、見出し、本文、図表キャプション等）
- ブロック単位のローカル翻訳エンジン（API不使用、差し替え可能）
- レイアウト近似再構成（文字サイズ調整・高さ拡張）
- 原文/翻訳プレビュー切替
- 翻訳結果PDFダウンロード
- LocalStorageへの中間データ保存・復元

## ローカル実行

静的ファイルなので任意のHTTPサーバーで起動してください。

```bash
python3 -m http.server 4173
```

その後 `http://localhost:4173` を開きます。

## 注意

- 初版はOCR非対応で、テキスト抽出可能PDFのみを正式対応としています。
- PDF出力の日本語フォントは環境差により見え方が変わる場合があります。
